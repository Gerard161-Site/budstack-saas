# Vice City Cannabis Template

## Overview

Vice City Cannabis is a bold, retro-inspired template that brings the vibrant aesthetics of 80s/90s street culture to modern medical cannabis dispensaries. With its comic-style artwork, neon color palette, and urban energy, this template is perfect for brands that want to stand out while maintaining professionalism.

## Key Features

- **Retro Comic Aesthetic**: Bold illustrations with vibrant color gradients
- **Neon Color Palette**: Coral (#FF6B6B), Teal (#4ECDC4), Yellow (#FFE66D)
- **Urban Typography**: Modern sans-serif fonts with bold headings
- **Scroll Animations**: Smooth transitions and parallax effects
- **Fully Responsive**: Mobile-first design optimized for all devices
- **Medical Compliance**: Professional layout that meets regulatory standards

## Customization

The template automatically adapts to your tenant's branding:

- **Business Name**: Displayed prominently throughout
- **Logo**: Optional overlay on hero section
- **Hero Image**: Full-width retro comic-style background
- **Colors**: Customizable via tenant settings
- **Typography**: Configurable font families
- **Content**: Dynamic page content from tenant settings

## Sections Included

1. **Hero Section**: Bold full-width hero with retro illustration
2. **About Section**: Team showcase with comic-style image
3. **Features Grid**: Key value propositions with icons
4. **Product Showcase**: Highlighted products with vibrant cards
5. **Consultation CTA**: Eye-catching call-to-action for bookings
6. **Footer**: Comprehensive navigation and contact info

## Technical Details

- **Framework**: Next.js 14+ with React 18+
- **Styling**: Tailwind CSS with custom color utilities
- **Animations**: Framer Motion for scroll effects
- **Icons**: Lucide React
- **Images**: Next.js Image optimization
- **Performance**: Lighthouse score 90+

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Installation

This template is designed to be uploaded via the BudStack Super Admin dashboard:

1. Push template to GitHub repository
2. Go to Super Admin → Templates
3. Click "Upload New Template"
4. Enter GitHub repository URL
5. Wait for upload to complete
6. Assign to tenant via Tenant Admin → Branding
